# Cheatsheet

This page is for quick recall while doing practical work.

## CRS Cheatsheet

- EPSG:4326: latitude and longitude for exchange and KML workflows.
- EPSG:3857: web map display CRS.
- EPSG:32643: projected metric CRS for analysis and measurements in this reference.

## Interoperability Cheatsheet

- CSV to Excel for cleaning and formulas.
- CSV to QGIS for spatial point loading.
- QGIS vector to CAD via mapimport.
- QGIS GeoTIFF to CAD via mapiinsert.
- CAD to GIS via mapexport.
- GIS to Google Earth Pro via KML or KMZ.
