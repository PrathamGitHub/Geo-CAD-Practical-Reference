# Further Reading

This page lists practical and trusted sources for deeper learning.

Use this page after finishing the core pages in this handbook.

## Official Product Documentation

- Autodesk Civil 3D 2021 Help.
- Microsoft Excel Help and Learning.
- QGIS User Guide (3.40).
- Google Earth Help Center.
- OneDrive Help and Learning.

## Standards and Format References

- OGC standard overview.
- EPSG coordinate system registry.
- GDAL tools and format behavior.

## Why These Sources Matter

- They are maintained by product or standards owners.
- They provide up-to-date behavior and limitations.
- They reduce trial-and-error in production workflows.

## Use This Reading Sequence

1. Complete [Core Concepts and Standards](concepts-and-standards.md).
2. Complete the tool pages in this site.
3. Run [Practical Execution Guide](practical-execution-guide.md).
4. Use official docs only for options not covered in this handbook.

## Suggested Reading Order

1. Read concepts and standards in this site first.
2. Use tool-specific reference pages in this site.
3. Use official pages for advanced options only when required.

## When to Use External Docs

- You need advanced options not in this handbook.
- A menu path changed in your software build.
- You need edge-case behavior for export/import formats.

If external guidance conflicts with this workshop workflow, keep CRS and unit consistency rules from this handbook as the final decision point.
