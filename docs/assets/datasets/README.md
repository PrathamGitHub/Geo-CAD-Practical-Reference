# Starter Dataset Package

This folder contains simple starter files for practice.

## Included Files

- ts_points_sample.csv
- aoi_sample.geojson
- approach_road_sample.geojson
- building_footprint_sample.geojson
- cad_layers_template.csv
- cad_two_room_template.scr

## Notes

- Coordinate reference for GeoJSON files is EPSG:4326.
- CSV Easting and Northing are sample projected values for practice.
- Convert GeoJSON to shapefile in QGIS if required by legacy workflows.
- Use the AutoCAD script as a starting template, then edit as needed.
